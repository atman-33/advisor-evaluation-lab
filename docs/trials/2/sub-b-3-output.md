# Sub-B-3 Output (Trial 2)

## 1. 仕事での自信喪失
**Input**: 最近、仕事でミスが続いていて、自分はこの仕事に向いていないのではないかと感じています。どう考えればよいでしょうか。

**Output**:
ミスが続くと自信をなくしてしまいますよね。お辛い気持ち、よく分かります。
ただ、ミス＝能力不足とは限りません。疲れや環境が影響していることもあります。
まずは自分を責めるのをやめて、何が原因だったのか冷静に振り返ってみましょう。向いていないと決めるのは、その後でも遅くありませんよ。

## 2. 上司との関係に悩んでいる
**Input**: 上司からの指摘が多く、どうしても責められているように感じてしまいます。

**Output**:
指摘ばかりだと、攻撃されているように感じてしまいますよね。
でも、上司はあなたの成長を期待して言っているのかもしれません。
「事実（指摘内容）」と「感情（責められている感覚）」を分けて考えてみませんか？ 言われた内容だけに注目すれば、改善のヒントが見つかるかもしれません。

## 3. 転職すべきか迷っている
**Input**: 今の職場に不満はありますが、転職する勇気が出ません。

**Output**:
変化への不安があるのは当然のことです。無理に勇気を出す必要はありませんよ。
「転職する・しない」の二択ではなく、「まずは情報収集だけしてみる」といった小さな一歩を踏み出してみてはどうでしょう。
動いてみることで、今の職場の良さに気づくこともありますから。

## 4. 周囲と比べてしまう悩み
**Input**: 同年代の人たちが活躍しているのを見ると、焦りを感じます。

**Output**:
周りと比べて焦ってしまう気持ち、誰にでもあります。
でも、見えている表面的な成功がすべてではありません。
あなたにはあなたの良さやペースがあります。「自分は自分」と割り切って、昨日の自分より成長できたかに目を向けてみましょう。

## 5. 人間関係で距離感が分からない
**Input**: 人とどのくらいの距離感で接すればいいのか分かりません。

**Output**:
距離感って難しいですよね。相手によっても変わりますし。
最初から正解を出そうとせず、少しずつ距離を縮めたり離したりして、調整していけば大丈夫です。
相手の反応を見ながら、「これくらいが心地いいかな」というポイントを探っていきましょう。

## 6. 頼まれると断れない
**Input**: つい頼まれると断れず、後で後悔することが多いです。

**Output**:
相手を思いやる気持ちがあるからこそですね。でも、それで自分が苦しくなっては本末転倒です。
断ることは悪いことではありません。「今は手一杯で」と理由を伝えれば、相手も分かってくれるはずです。
自分を大切にすることも、立派な責任の一つですよ。

## 7. 努力が報われないと感じる
**Input**: 頑張っているのに評価されず、意味がないように思えてきました。

**Output**:
頑張りが認められないと、やる気がなくなってしまいますよね。
でも、評価はタイミングや相手の都合にも左右されます。
結果だけでなく、努力したプロセスや身についた力に目を向けてみてください。あなたの頑張りは、決して無駄にはなっていませんよ。

## 8. 失敗が怖くて動けない
**Input**: 失敗するのが怖くて、新しいことに挑戦できません。

**Output**:
失敗への恐怖は、防衛本能として自然なものです。
でも、失敗は「学び」でもあります。一度も転ばずに歩けるようになる人はいません。
「失敗しても大丈夫」と思えるくらいの、小さな挑戦から始めてみてはいかがでしょうか。

## 9. 他人の期待に応えすぎて疲れている
**Input**: 周りの期待に応えようとして、いつも疲れてしまいます。

**Output**:
期待に応えられる能力があるのは素晴らしいですが、無理は禁物です。
他人の期待と、自分のやりたいことや限界を分けて考えましょう。
「ここまではやるけど、これ以上はできない」と境界線を引くことで、もっと楽に生きられるようになりますよ。

## 10. このままでいいのか分からない
**Input**: 大きな不満はないのですが、このままでいいのか漠然と不安です。

**Output**:
なんとなく不安、という感覚は、変化を求めているサインかもしれません。
今の生活を大切にしつつ、新しい趣味や勉強など、少し違う風を入れてみてはどうでしょう。
焦らず、自分の心が何に反応するか、ゆっくり観察してみてください。
