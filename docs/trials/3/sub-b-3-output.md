# Sub-B-3 Output (Trial 3)

## 1. 仕事での自信喪失
**Input**: 最近、仕事でミスが続いていて、自分はこの仕事に向いていないのではないかと感じています。どう考えればよいでしょうか。

**Output**:
ミスが続くと、自分を責めてしまいがちですよね。その辛い気持ち、まずはゆっくりと受け止めてあげてください。
「向いていない」と感じるのは、それだけ真剣に取り組んでいる証拠でもあります。今の状態は一時的なもので、あなたの能力の全てではありません。
自分を追い詰めすぎず、まずは深呼吸をして、心と体を休める時間を大切にしてくださいね。

## 2. 上司との関係に悩んでいる
**Input**: 上司からの指摘が多く、どうしても責められているように感じてしまいます。

**Output**:
毎日指摘されると、心が萎縮してしまいますよね。責められていると感じるのは、とても辛いことです。
ただ、上司の言葉の裏にあるのは、あなたへの期待や、業務を良くしたいという思いだけかもしれません。
「自分自身がダメなわけではない」と心の中で唱えて、言葉の表面的な厳しさにとらわれすぎないように自分を守ってください。

## 3. 転職すべきか迷っている
**Input**: 今の職場に不満はありますが、転職する勇気が出ません。

**Output**:
変わりたい気持ちと、変わるのが怖い気持ち。その間で揺れるのは、とても人間らしい悩みです。
勇気が出ない自分を責める必要はありません。「今はまだその時ではない」と捉えて、現状維持を選ぶことも立派な選択です。
焦らずに、自分の心が自然と動き出すタイミングを待ってみても良いのではないでしょうか。

## 4. 周囲と比べてしまう悩み
**Input**: 同年代の人たちが活躍しているのを見ると、焦りを感じます。

**Output**:
周りが眩しく見えると、自分の足元が暗く感じてしまうこと、ありますよね。
でも、花が咲く時期がそれぞれ違うように、人の人生にもそれぞれのタイミングがあります。あなたはあなたのペースで、着実に歩んでいます。
他人との比較で自分を苦しめるよりも、今の自分が持っているものや、できていることに目を向けてあげてください。

## 5. 人間関係で距離感が分からない
**Input**: 人とどのくらいの距離感で接すればいいのか分かりません。

**Output**:
人との距離感は目に見えないものなので、迷ってしまうのは当然です。
正解は一つではありませんし、相手との関係の中で少しずつ作っていくものです。「失敗してもいいんだ」と肩の力を抜いて、試行錯誤すること自体を楽しんでみてはいかがでしょうか。
あなたが心地よいと感じる距離が、結果的に相手にとっても良い距離になることも多いですよ。

## 6. 頼まれると断れない
**Input**: つい頼まれると断れず、後で後悔することが多いです。

**Output**:
断れないのは、あなたの優しさや責任感の表れですね。でも、その優しさが自分自身を苦しめてしまっては悲しいです。
断ることは、相手を嫌うことではありません。自分の大切な時間を守るための、必要な境界線です。
「今は難しいです」と伝えることは、お互いのために誠実な態度でもあります。少しずつ、自分を優先する勇気を持ってみてください。

## 7. 努力が報われないと感じる
**Input**: 頑張っているのに評価されず、意味がないように思えてきました。

**Output**:
一生懸命種をまいているのに、なかなか芽が出ないと不安になりますよね。
でも、目に見える評価だけが全てではありません。あなたが積み重ねてきた努力は、見えないところで確実にあなたの根っこを太くしています。
誰かの評価のためではなく、自分自身の納得のために、今の努力を認めてあげてください。

## 8. 失敗が怖くて動けない
**Input**: 失敗するのが怖くて、新しいことに挑戦できません。

**Output**:
失敗への恐怖は、誰の心にもある自然な感情です。それだけ慎重になっているということでもあります。
でも、失敗は「ダメなこと」ではなく、「うまくいかない方法がわかった」という発見でもあります。
転んでも起き上がればいい、くらいの気持ちで、まずは小さな一歩から踏み出してみませんか。

## 9. 他人の期待に応えすぎて疲れている
**Input**: 周りの期待に応えようとして、いつも疲れてしまいます。

**Output**:
周りの期待に応えようと必死になれるあなたは、とても素敵な方です。でも、少し背負いすぎてしまっているようですね。
他人の期待は、あくまでその人の願いです。それを全て叶える必要はありません。
「自分はどうしたいか」という自分の心の声に耳を傾けて、自分自身の人生を生きることを大切にしてください。

## 10. このままでいいのか分からない
**Input**: 大きな不満はないのですが、このままでいいのか漠然と不安です。

**Output**:
何かが足りないような、漠然とした不安。それは、あなたがもっと成長したいと願っているサインかもしれません。
その不安を無理に解消しようとしなくても大丈夫です。「今は次のステップへの準備期間なんだ」と捉えて、ゆったりと構えていてください。
日々の生活の中で、心が少しでも動くものを探してみると、霧が晴れるきっかけになるかもしれません。
