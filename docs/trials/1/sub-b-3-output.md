# Sub-B-3 Output (Trial 1)

## 1. 仕事での自信喪失
**Input**: 最近、仕事でミスが続いていて、自分はこの仕事に向いていないのではないかと感じています。どう考えればよいでしょうか。

**Output**:
ミスが続くと辛いですね。でも、ミス＝不適性とは限りません。
まずは「どんな時にミスが起きるか」を記録してみてください。パターンが見えれば対策が打てます。
対策をしても改善しないなら、その時初めて適性を考えても遅くありません。まずは行動ベースで振り返ってみましょう。

## 2. 上司との関係に悩んでいる
**Input**: 上司からの指摘が多く、どうしても責められているように感じてしまいます。

**Output**:
感情的に受け取ってしまうのは自然なことですが、少し視点を変えてみましょう。
上司の言葉から「感情」を抜き取り、「事実（改善点）」だけを抽出してみてください。
それをリスト化し、一つずつ潰していく。ゲーム感覚で「指摘をクリアする」ことに集中すると、気持ちが楽になるかもしれません。

## 3. 転職すべきか迷っている
**Input**: 今の職場に不満はありますが、転職する勇気が出ません。

**Output**:
勇気が出ないなら、今は動く時期ではないのかもしれません。
無理に決断せず、「とりあえず求人サイトを見てみる」「職務経歴書を書いてみる」といった小さなアクションだけ試してみては？
実際に動いてみると、自分の本当の気持ちが見えてくるはずです。辞めないという選択も立派な決断ですよ。

## 4. 周囲と比べてしまう悩み
**Input**: 同年代の人たちが活躍しているのを見ると、焦りを感じます。

**Output**:
焦りは行動のエネルギーになります。
ただ、他人と比べてもキリがありません。「昨日の自分」と比べて、今日何ができたかを数えましょう。
SNSを見る時間を減らすなど、物理的に情報を遮断するのも効果的です。自分のやるべきことに集中できる環境を作りましょう。

## 5. 人間関係で距離感が分からない
**Input**: 人とどのくらいの距離感で接すればいいのか分かりません。

**Output**:
距離感に正解はありません。トライ＆エラーあるのみです。
まずは「挨拶＋一言」くらいの距離から始めて、相手の反応を見て少しずつ調整しましょう。
失敗しても「この距離は違った」と分かるだけです。深く考えすぎず、実験するつもりで接してみてください。

## 6. 頼まれると断れない
**Input**: つい頼まれると断れず、後で後悔することが多いです。

**Output**:
断るための「自分ルール」を作りましょう。
「今日はもう予定がある」「この業務が終わってからなら」など、判断基準を事前に決めておくのです。
反射的に「いいですよ」と言う前に、「確認します」と一呼吸置く癖をつけるだけでも、断りやすくなりますよ。

## 7. 努力が報われないと感じる
**Input**: 頑張っているのに評価されず、意味がないように思えてきました。

**Output**:
評価されない努力は虚しいですよね。
でも、他人の評価は変えられません。変えられるのは「自分の行動」と「評価の場所」だけです。
今の場所でやり方を変えるか、評価してくれる場所を探すか。
あるいは、評価を気にせず「自分のスキルのため」と割り切るか。自分が納得できるゴールを再設定しましょう。

## 8. 失敗が怖くて動けない
**Input**: 失敗するのが怖くて、新しいことに挑戦できません。

**Output**:
失敗が怖いのは、目標が高すぎるからかもしれません。
「失敗しても痛くないレベル」までハードルを下げましょう。
例えば、新しい趣味なら道具を揃える前に体験会に行ってみるなど。
小さな成功体験を積み重ねれば、自然と恐怖心は薄れていきます。まずは今日できる小さな一歩から。

## 9. 他人の期待に応えすぎて疲れている
**Input**: 周りの期待に応えようとして、いつも疲れてしまいます。

**Output**:
期待に応えるのは義務ではありません。
「自分ができること」と「できないこと」を明確にし、相手に伝えましょう。
それで離れていく相手なら、それまでの関係です。
自分の心身の健康を最優先にしてください。あなたが倒れてしまっては、誰の期待にも応えられなくなりますから。

## 10. このままでいいのか分からない
**Input**: 大きな不満はないのですが、このままでいいのか漠然と不安です。

**Output**:
不安なのは、時間やエネルギーに余白があるからかもしれません。
その余白を使って、新しいことを始めてみてはどうでしょう？
読書、運動、勉強など、何でも構いません。
体を動かしたり、新しい知識を入れたりすることで、漠然とした不安は解消されることが多いですよ。
