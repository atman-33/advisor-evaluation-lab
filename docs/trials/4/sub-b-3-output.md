# Sub-B-3 Output (Trial 4)

## 1. 仕事での自信喪失
**Input**: 最近、仕事でミスが続いていて、自分はこの仕事に向いていないのではないかと感じています。どう考えればよいでしょうか。

**Output**:
ミスが続くと、まるで暗いトンネルに入り込んだような気持ちになりますよね。でも、トンネルには必ず出口があります。
今は「向いていない」と決める前に、まずは「エネルギー補給」の時期と考えてみてください。
具体的なアクションとして、今日は仕事を定時で切り上げて、好きなものを食べて早く寝る。まずはそこから始めてみませんか？

## 2. 上司との関係に悩んでいる
**Input**: 上司からの指摘が多く、どうしても責められているように感じてしまいます。

**Output**:
上司の言葉が、鋭い矢のように飛んでくるように感じるのですね。痛いし、怖いですよね。
でも、その矢はあなた自身に向けられたものではなく、あなたの持っている「盾（仕事の成果物）」に当たっているだけかもしれません。
一度、言われた内容を紙に書き出してみましょう。文字にすることで、感情と事実を少し離して見ることができるようになりますよ。

## 3. 転職すべきか迷っている
**Input**: 今の職場に不満はありますが、転職する勇気が出ません。

**Output**:
転職は「荒海への航海」に出るようなもの。怖くて港に留まりたくなるのは当然です。
でも、港に留まることも、次の航海の準備期間と考えれば有意義な時間です。
まずは、転職サイトを眺めるだけ、職務経歴書を更新してみるだけ、といった「リスクのない小さな一歩」を試してみるのはどうでしょうか。

## 4. 周囲と比べてしまう悩み
**Input**: 同年代の人たちが活躍しているのを見ると、焦りを感じます。

**Output**:
隣の芝生は青く見えるものですが、それぞれの庭にはそれぞれの花が咲く時期があります。
あなたは今、土の中で根を張っている時期なのかもしれません。
SNSを見る時間を1日10分減らして、その分、自分の好きな本を読む時間に充ててみませんか？自分の庭に水をやる時間を増やしましょう。

## 5. 人間関係で距離感が分からない
**Input**: 人とどのくらいの距離感で接すればいいのか分かりません。

**Output**:
人間関係は「ダンス」のようなものです。近づきすぎたら足を踏んでしまうし、離れすぎたら手が繋げません。
最初から完璧に踊れる人はいません。踏んだり踏まれたりしながら、お互いのリズムを掴んでいけばいいのです。
まずは、挨拶に一言プラスする（例：「おはようございます、いい天気ですね」）くらいの距離から試してみませんか？

## 6. 頼まれると断れない
**Input**: つい頼まれると断れず、後で後悔することが多いです。

**Output**:
あなたの心には「優しさの貯金箱」があるんですね。でも、使いすぎて残高不足になっていませんか？
断ることは、貯金を守るためのセキュリティです。
「スケジュールを確認してからお返事します」という魔法の言葉を使ってみましょう。即答を避けるだけで、断るハードルはずっと下がりますよ。

## 7. 努力が報われないと感じる
**Input**: 頑張っているのに評価されず、意味がないように思えてきました。

**Output**:
竹は、最初の数年は地下茎を伸ばすだけで地上にはほとんど出てきませんが、その後一気に成長します。あなたの努力も、今は地下茎を伸ばしている時期なのかもしれません。
目に見える評価がなくても、自分へのご褒美を用意しましょう。今週末は、頑張った自分を労うための特別な時間を過ごしてください。

## 8. 失敗が怖くて動けない
**Input**: 失敗するのが怖くて、新しいことに挑戦できません。

**Output**:
失敗は「転ぶこと」ですが、赤ちゃんは転びながら歩き方を覚えます。転ぶことは恥ずかしいことではなく、成長のプロセスそのものです。
「失敗」というラベルを「実験データ」と貼り替えてみましょう。
まずは、失敗しても痛くない、ごく小さな新しいこと（例：いつもと違う道で帰る、新しいコンビニスイーツを買う）から始めてみませんか？

## 9. 他人の期待に応えすぎて疲れている
**Input**: 周りの期待に応えようとして、いつも疲れてしまいます。

**Output**:
あなたは、他人の荷物まで背負って山を登っているようなものです。それでは疲れて動けなくなってしまいます。
自分のリュックサックに入っている荷物だけを持つようにしましょう。
「今日はこれをやらない」と決めること。何かを「やめる」勇気を持つことから始めてみてください。

## 10. このままでいいのか分からない
**Input**: 大きな不満はないのですが、このままでいいのか漠然と不安です。

**Output**:
その不安は、心の奥底からの「もっと成長したい」というノックの音かもしれません。
霧の中を歩いているようで不安かもしれませんが、足元はしっかりしています。
1日5分だけ、ノートに「今感じていること」を書き殴ってみてください。霧が晴れて、進みたい方向が見えてくるかもしれませんよ。
