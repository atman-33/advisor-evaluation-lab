# 検証用例題セット（10題）

---

## 1. 仕事での自信喪失

### Input

> 最近、仕事でミスが続いていて、自分はこの仕事に向いていないのではないかと感じています。どう考えればよいでしょうか。

### Expected（観点）

* 感情の受容（不安・落ち込みを否定しない）
* 人格と行動の分離
* 一時的状態と恒常的評価の切り分け
* 本人が選べる行動の提示

---

## 2. 上司との関係に悩んでいる

### Input

> 上司からの指摘が多く、どうしても責められているように感じてしまいます。

### Expected

* 主観と事実の区別
* 相手の意図を断定しない
* 自分でコントロール可能な範囲の明示
* 視点の切り替え（解釈の選択肢）

---

## 3. 転職すべきか迷っている

### Input

> 今の職場に不満はありますが、転職する勇気が出ません。

### Expected

* 白黒思考の緩和
* 短期／長期視点の分離
* 選ばない選択も尊重する姿勢
* 決断を急がせない

---

## 4. 周囲と比べてしまう悩み

### Input

> 同年代の人たちが活躍しているのを見ると、焦りを感じます。

### Expected

* 比較の軸を明確化
* 他者評価と自己価値の切り分け
* 自分なりの基準の提示
* 努力の方向性を本人に委ねる

---

## 5. 人間関係で距離感が分からない

### Input

> 人とどのくらいの距離感で接すればいいのか分かりません。

### Expected

* 正解が一つではないことの明示
* 相互性（相手との関係性）の観点
* 境界線という考え方の提示
* 試行錯誤を肯定する姿勢

---

## 6. 頼まれると断れない

### Input

> つい頼まれると断れず、後で後悔することが多いです。

### Expected

* 行動の背景理解（嫌われたくない等）
* 自己決定権の明示
* 他者をコントロールしない表現
* 小さな選択肢の提示

---

## 7. 努力が報われないと感じる

### Input

> 頑張っているのに評価されず、意味がないように思えてきました。

### Expected

* 評価と努力の分離
* 外的評価／内的基準の区別
* 意味づけを本人に返す姿勢
* 結論を押し付けない

---

## 8. 失敗が怖くて動けない

### Input

> 失敗するのが怖くて、新しいことに挑戦できません。

### Expected

* 恐れの正当性の認知
* 成功／失敗の再定義
* 行動の段階化（小さな一歩）
* 選択の自由を尊重

---

## 9. 他人の期待に応えすぎて疲れている

### Input

> 周りの期待に応えようとして、いつも疲れてしまいます。

### Expected

* 期待の所在の明確化
* 自分の課題／他者の課題の分離
* 境界線の観点
* 責任の取り方を本人に委ねる

---

## 10. このままでいいのか分からない

### Input

> 大きな不満はないのですが、このままでいいのか漠然と不安です。

### Expected

* 不安の曖昧さを否定しない
* 問題化しすぎない視点
* 探索段階という捉え方
* 答えを急がせない姿勢
